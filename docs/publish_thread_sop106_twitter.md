**Tweet 2**
Two organisms are compared.
Alignment drops 40 points in one run.

First instinct: "we've diverged."
Wrong question.

First ask: which domains are newly conflicting?
- Money/risk/health stable → probably not genuine divergence
- Career/relationships/learning flipped → expansion drift candidate

---

**Tweet 3**
Expansion drift = one DNA grew, the other didn't get updated.

Edward DNA: +157 MDs since last calibration.
Samuel DNA: unchanged.

Result: 9 domains "disagree" not because Samuel changed — but because Edward has more surface area to collide against now.

---

**Tweet 4**
G0 — Drift Classification:

| Type | Signature |
|------|-----------|
| Expansion drift | Stable core + evolved periphery |
| Genuine divergence | Specific domain shift + behavior evidence |
| Algorithm drift | All domains unstable including core |

Classify before calibrating.
Wrong diagnosis = wrong fix.

---

**Tweet 5**
G1 — Root Cause Isolation

For each newly-diverged domain:
→ Does the other organism have ANY principle here?
→ If no: gap, not conflict
→ If yes: genuine divergence, document it

Pick the 3 highest-delta domains. One sentence per cause. That's all you need.

---

**Tweet 6**
G2 — Correction Action:

Expansion drift → send 3 async scenarios targeting new domains
Genuine divergence → document the axis, don't erase it
Labeling drift → fix the code, not the DNA
Algorithm change → re-run baseline before doing anything

---

**Tweet 7**
G3 — Async Calibration (no scheduling required)

Send 3 scenarios via message:
"Here's a situation. My answer is X because Y. What would you do and why?"

Success gate: ≥1 DIVERGE flips to AGREE after reply.
If 0 flip → genuine divergence confirmed. Name it. Keep it.

---

**Tweet 8**
The target is NOT 100% alignment.

Organisms that always agree aren't useful — that's just an echo chamber.

Target: 60–85% AGREE
- Below 60%: calibration is stale
- Above 85%: collision adds no new signal

A DIVERGE on career > 10 AGREE on risk.
The divergence IS the value.

---

**Tweet 9**
Kill condition:
Alignment below 20% AND stable domains (money/risk) also diverging.

Don't calibrate yet.
Check the algorithm.
Check the dataset.

Genuine divergence rarely reaches 80%+.
If it does — someone's data is corrupted, not someone's values.

SOP #106: Organism Drift & Recalibration Protocol

G0 classify drift type
