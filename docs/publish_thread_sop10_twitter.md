# Twitter Thread — SOP #10: Decision-Making Under Uncertainty

**Draft** | Ready to post | 10 tweets

**Tweet 1 (hook)**
Most people make bad decisions under uncertainty because they're reading the wrong signal.

Not wrong data. Wrong *type* of data.

Here's the 5-gate framework I use for every high-stakes decision (trading, career, negotiation):

---

**Tweet 2**
Gate 0: Read the DERIVATIVE, not the level.

"Where is it now?" → stale information.
"Where is it going, and how fast?" → edge.

If your analysis only describes current state, you have no edge. You're reacting, not anticipating.

---

**Tweet 3**
Gate 1: Name your information asymmetry.

Price = sufficient statistic of all *public* info.
Public data → zero edge by definition.

You need structural, temporal, or interpretive asymmetry.
If you can't name it, you don't have it.

---

**Tweet 4**
Structural edge: data others can't access.
Temporal edge: same data, processed faster.
Interpretive edge: same data, better model.

Skin-in-the-game creates structural edge.
Regime detection creates temporal edge.
Mental models above the population create interpretive edge.

---

**Tweet 5**
Gate 2: Escalate to meta-strategy if stuck.

Object level: "Should I adjust parameters?"
Meta level: "Am I even using the right strategy for this regime?"

Rule: 3 cycles of object-level iteration without improvement → escalate one level up.

---

**Tweet 6**
Gate 3: Check population behavior.

What is the consensus action?
Does the opposite have positive EV?

Contrarian ≠ edge.
Contrarian WITH a structural reason for population error = edge.

The question is always: why are they wrong?

---

**Tweet 7**
Gate 4: Default to inaction.

No edge = negative EV action.
No action = preserve optionality.

"Doing nothing feels wrong" is loss-aversion disguised as conviction.

Inaction ≠ not thinking. Accumulate information. Hold no position.

---

**Tweet 8**
These 5 gates map directly to 5 core principles:

1. Rate of change > level
2. Information asymmetry → action direction
3. Meta-strategy manages strategy
4. Population exploit for contrarian EV
5. Inaction as default without edge

They work in trading. They work in life.
