# Engagement Response Templates

> Purpose: paste-ready replies for common scenarios. Zero in-the-moment decisions.
> Check X once at 16:00. Pick template, personalize 1 sentence if needed, send.

---

## Category 1 — Positive/Agreement

**Scenario: "This is exactly what I needed"**
```
Glad it landed. Which gate was the most useful for where you're at?
```

**Scenario: "Great thread" / "Nice"**
```
Appreciate it. Thread #02 covers portfolio construction — same gate framework. Worth the read if you're at the sizing stage.
```

---

## Category 2 — Question / Challenge

**Scenario: "What if the market regime changes mid-strategy?"**
```
That's Gate 3 in SOP #07 (regime detection). Short answer: regime change triggers rotation, not stop-loss. Thread coming in the queue — or DM me if you want the pre-release PDF.
```

**Scenario: "Does this work for [specific market/asset]?"**
```
The framework is market-agnostic — it's about EV structure, not instrument. The friction cost calc in Gate 1 will look different for [their market] but the logic holds. What market are you on?
```

**Scenario: Technical pushback / "That's not how it works"**
```
Fair question. Walk me through your setup — specifically what breaks in Gate [X]. Happy to steelman the countercase.
```

## Category 3 — DMs (pre-offer, G0 trust-building)

**DM #1 (first contact): "How do I get started with this?"**
```
Depends on where you are. Are you: (A) still backtesting / developing a strategy, (B) live but bleeding, or (C) profitable but wanting to systematize?
```

**DM #2 (after they answer):**
```
Got it. [Gate reference from SOP that matches their situation]. The full framework is in the thread series — SOP #[X] covers this directly. What's the specific decision you're stuck on?
```

**DM #3 (after 3 exchanges — G0 threshold met, offer eligible):**
```
This is actually something I work through 1:1. I do a 60-min session where we map your strategy to the gate framework and find the leaks. If that's useful, I can share how it works.
```

**If G0 met (≥ 3 substantive exchanges):**
```
Yes, I do consulting sessions. $[500]/hr, 60 min, systematic trading frameworks. Focused on: strategy validation, risk architecture, or sizing logic. Send me the 1-sentence problem statement and I'll confirm fit.
```
