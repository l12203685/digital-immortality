# Economic Autonomy Roadmap
> Branch 3 — Build cash-flow loops that cover agent running costs

**Core constraint:** Agent API costs ~$20-50/month (Sonnet 4.6, daemon cycles). Target: ≥NT$3,000/month passive to break even. FIRE-compatible = income flows without requiring Edward's active time.

---

## Ranked Opportunities

### Tier 1 — Highest EV, Lowest Friction

#### 1. Samuel Consulting (Decision Systems)
- **Format:** 1:1 or small group sessions, applying EV-based decision frameworks to novel problems
- **Pricing:** NT$3,000-8,000/session × 2/month = NT$6,000-16,000/month
- **Edge:** Edward has working system (18/18 accuracy). Samuel has network + distribution. Combined = real product.
- **First step:** Define 3 use cases where Edward's kernel solves a problem someone will pay for (career pivot, investment timing, relationship boundary decisions)
- **oos_4 prediction:** CONDITIONAL YES — 1-2 cases first, no long-term commitment
- **Risk:** Time cap must hold. Cap at 2 sessions/month max.

#### 2. DNA Calibration as a Service (B2C)
- **Format:** Async. Client answers 20 scenario questions. Edward (or trained agent) returns calibrated decision profile + 3 blind spots.
- **Pricing:** NT$5,000-10,000/report (one-time)
- **Edge:** Nobody else has a working behavioral-fidelity test system. Consistency test is already built.
- **First step:** Package `consistency_test.py` + scenario bank into a client-facing intake form. Write one sample report.
- **Risk:** Requires Edward to review reports. Not fully passive until agent handles full pipeline.

### Tier 2 — Medium EV, Moderate Friction

#### 3. Method Documentation Licensing
- **Format:** ZP/ methodology → packaged course or document set
- **Pricing:** NT$500-2,000/license (passive after creation)
- **Edge:** Zeroth Principles is already built. ZP = public methodology repo.
- **First step:** Identify 3 ZP chapters with highest standalone value. Convert to standalone PDF product.
- **Risk:** Distribution hard without an audience. Requires cold audience building.

#### 4. API Agent Subscription (Tech-Forward)
- **Format:** Small businesses pay for access to an Edward-style decision agent (custom DNA optional)
- **Pricing:** NT$1,500-3,000/month/client × 5 clients = NT$7,500-15,000/month
- **Edge:** Platform already works. Daemon runs autonomously.
- **First step:** Define minimum viable client interface. What can clients actually ask? What's the boundary?
- **Risk:** Legal/liability if advice is wrong. Requires careful scoping.
