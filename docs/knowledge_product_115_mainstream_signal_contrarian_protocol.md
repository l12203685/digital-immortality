# SOP #115 — 主流信號逆向判讀協議 (Mainstream Signal Contrarian Protocol)

> UTC: 2026-04-10T19:30:00+00:00
> Backing MDs: MD-398 (主流媒體×家族群組=散戶進場訊號), MD-397 (理解先行=資本配置門檻)
> Domain: Market timing / Contrarian indicators / Risk management
> Decision label: READ_SIGNAL_LAYER_BEFORE_ACTING

---

## 觸發條件

任何以下一項成立即觸發本 SOP：
- 非金融圈朋友主動問你某資產「現在還能買嗎」
- 家族群組或同學群組出現某資產的分享/轉傳
- 主流電視/新聞媒體開始密集報導某投資主題
- 社交媒體演算法開始主動推送某資產相關內容給非投資帳號

---

## G0 — 信息擴散層次校準

**目的**：確定你在資訊擴散鏈上的位置，決定當前是優勢期還是劣勢期。

| 熱度層級 | 定義 | 你的資訊優勢 |
|---------|------|------------|
| L0 | 只有特定社群/早期採用者知道 | 最高 — 買方稀缺，未來增量大 |
| L1 | 科技/金融小圈子在討論 | 高 — 主流還未跟上 |
| L2 | 科技媒體報導，非金融朋友開始問 | 中 — 主流進場中 |
| L3 | 電視台報導 + 家族群組轉傳 | 低/無 — 最後一批散戶進場 |
| L4 | 計程車司機/長輩主動建議買 | 負 — 有信息優勢的人已在出場 |

**校準動作**：
1. 對照上表，判斷當前資產的熱度層級
2. 如果你在 L0 時建倉，現在到了 L3/L4，你的 exit window 已開始縮小
3. 如果你在 L3/L4 時才聽說，不要把「別人的熱情」誤認為「自己的信息優勢」

**PASS 條件**：確認當前熱度層級 → 進 G1

---

## G1 — 持倉暴露度審計

**目的**：評估你目前在該資產的暴露，並對照熱度層級調整。

| 熱度層級 | 建議操作 |
|---------|---------|
| L0-L1 | 正常持倉，按既定 thesis 管理 |
| L2 | 停止加碼，開始制定退出節點 |
| L3 | 分批出場窗口，設定觸發條件（不是「看到泡沫破才出」） |
| L4 | 立即評估全部或大部分出場 |

**退出節點設定（L3 標準）**：
```
退出條件 = 滿足以下任一：
- 價格從峰值回落 X%（根據歷史波動率設定，加密資產建議 20-30%）
- 主要 thesis 假設被 falsified（e.g. 監管改變/基本面惡化）
- 更好的資本配置機會出現
```

**PASS 條件**：退出節點已書面化 → 進 G2

---

## G2 — 逆向信號確認

**目的**：區分「真正的新訊息」和「主流信息滯後傳播」。

主流媒體報導的內容，90%+ 是市場已經 price-in 的信息。問自己：

1. **這個報導告訴了我什麼我之前不知道的？**
   - 有新信息 → 更新 thesis
   - 沒有新信息 → 這是散戶進場，不是你的信息優勢

2. **這個熱度是因為基本面改善，還是因為資金追漲？**
   - 基本面驅動（e.g. 技術突破/收益改善）→ L3 熱度可能仍有空間
   - 資金驅動（e.g. 價格漲所以更多人進來看）→ 正反饋泡沫的典型特徵

3. **「現在進場」的人，他們的 EV 是正的嗎？**
   - 主流散戶在 L3 進場的歷史勝率極低
   - 你若在 L0 進場，出場給他們是 EV+ 交易

**PASS 條件**：確認這是資金驅動熱度，非新基本面 → 進 G3

---

## G3 — 執行退出計畫

**目的**：按 G1 書面化的退出節點執行，不因情緒改變計畫。

```
退出執行紀律：
1. 分批出場（避免一次出清造成滑點+情緒壓力）
   - e.g. 第1批 25% @ 現價，第2批 25% @ -10%，第3批 50% @ 退出觸發條件
2. 不等「最高點」出場（最高點只有事後才知道）
3. 出場後記錄：出場原因 / 當時熱度層級 / 回測：熱度層級對這次預測是否有效
4. 若出場後價格繼續漲，記錄「這次 L3 後繼續漲了 X%」作為下次校準數據
```

**PASS 條件**：出場計畫已執行或出場觸發條件已書面追蹤 → 進 G4

---

## G4 — 空倉期部署

**目的**：出場後，資本去哪裡？

| 選項 | 適用條件 |
|-----|---------|
| 現金/低風險資產 | 沒有更高 EV 機會，等下一個 L0-L1 信號 |
| 其他資產 L0-L1 | 有更好的信息優勢機會 |
| 回補原資產 | 若原資產回落到 thesis 重新成立的價格 |

**L0-L1 資產的篩選標準（逆向搜尋）**：
- 主流媒體「從不報導」或「嘲諷」的領域
- 你的非金融朋友圈對此毫無意識
- 有基本面但沒有炒作敘事

**PASS 條件**：資本再部署計畫已書面化 → G4 完成

---

## G5 — Kill 條件

本 SOP 的無效化條件（這些情況下熱度層級不適用）：
- 政策強制變化（監管/稅制）使所有持有者面臨非市場因素強制出場 → 不是散戶進場，是外生風險事件，用不同框架
- 技術革命性突破讓 L3 熱度由真實需求驅動（罕見，需要 thesis 重新建立）
