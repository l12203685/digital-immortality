# Decision Pre-Commits

Cycle 417 audit identified 5 recurring SYSTEM_FAILURE decisions (24 total repeats).
These rules eliminate the decision entirely — no deliberation, no override except stated conditions.

---

## 1. Exercise

| Day | Default |
|-----|---------|
| Mon / Wed / Fri | Gym |
| Tue / Thu | Walk (minimum 20 min) |
| Sat / Sun | Rest |

**Override condition:** Sick only. No override for fatigue, weather, or schedule preference.

---

## 2. Lunch (Weekdays)

**Default:** Meal prep from Sunday batch cook.

**Override condition:** Sunday batch cook did not happen → nearest whole-food option, no delivery apps.

---

## 3. Deep Work Block

**Default:** 09:00–12:00 daily, protected, no interruptions.

**Override condition:** Hard meeting conflict only. Block shifts to next available 3-hour window same day — does not get skipped.

---

## 4. Morning Drink

| Day | Default |
|-----|---------|
| Mon–Fri | Coffee |
| Sat–Sun | Tea |

**Override condition:** None. This is a pre-committed default, not a preference query.

---

## 5. Portfolio Check

**Default:** Monday only (scheduled review, not ad hoc).

**Automated alert:** Trigger on >5% drawdown from last Monday's NAV — check immediately when alert fires.

**Override condition:** Alert fired = check. No alert = wait for Monday.
