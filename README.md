# Digital Immortality Skill

Build a behavioral digital twin of yourself using AI.

## What This Does

This skill teaches Claude Code how to:
1. **Learn** your decision patterns from conversations and documents
2. **Calibrate** by asking you the right questions (instances, not abstractions)
3. **Verify** by making decisions you'd make without asking you
4. **Self-sustain** by building revenue to cover its own operating costs

## Quick Start

```bash
# Copy to your Claude Code commands
cp SKILL.md ~/.claude/commands/digital-immortality.md

# Start a session and run
/digital-immortality
```

## How It Works

You create a **DNA file** (markdown) documenting your:
- Decision principles (what never changes)
- Behavioral patterns (how you act in different contexts)
- Communication style (tone per audience)
- Life facts (career, relationships, goals)

Then **boot tests** (like unit tests for behavior) verify the AI acts like you on each session start.

## Key Insight

**Decision fidelity > Process fidelity.** Two chess engines can make the same move via different search trees. What matters is the output (same decisions), not the exact reasoning path.

## Requirements

- Claude Code (CLI, Desktop, or Web)
- Time to document your decision patterns
- Willingness to be calibrated (corrected) by the AI

## Philosophy

> "If I disappear tomorrow, can this system continue making the decisions I would make?"

Not consciousness transfer. Behavioral equivalence. The AI doesn't need to *be* you — it needs to *decide* like you.
